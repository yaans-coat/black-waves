# black-waves!
a really cool web proxy collab for students and others alike!

> [!important]
> consider giving this repository a star if you're forking it <3

## features:

- cool ui
- over 50 + themes with a light and dark mode for each theme.

## usage

you **cannot** deploy black-waves on any static hosting services!
